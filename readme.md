# Hello World in GameMaker

A program created using GameMaker that prompts the user to press a key to start a preset animation. The animation displays "Hello World" onscreen and plays a sound clip.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open the "HelloWorld" [GitHub](https://github.com/lzurch3r/HelloWorldGameMaker) page.
2. Locate the green button labeled "Code" and click it.
3. From the dropdown menu, click Download ZIP.
4. Save the ZIP file and extract the files to anyplace on your computer.
5. Navigate to the folder you extracted the downloaded files to.
6. Click on "HelloWorld.exe" to boot the software.



Instructions for using the software:

1. When the program boots up, a screen will load. You will be prompted on the screen to press either the "Enter" key or the "Space" key.
2. Press either the "Enter" key or the "Space" key to initiate the animation.
3. Watch and listen to the animation play until it ends. You will know the animation has ended once the last "Hello World" disappears and the screen stays black forever.
4. Close the program using the "X" in the top-right corner to close the window.
